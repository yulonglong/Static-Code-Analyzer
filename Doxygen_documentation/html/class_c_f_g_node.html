Detailed Description

A set of methods to extract more relationships after the CodeParser has parsed the SIMPLE source program.

Overview: CFGNode is the node structure of CFG nodes. This is a derived class that inherits the Node Class. In addition to the orginal Node Class's public and private attributes, it has a LIST of multiple parents. Also, it includes the corresponding getters and setters for the multiParent attribute.
